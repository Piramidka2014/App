телеграмм бот: Covid in the world
Позволяет узнать последние данные по заболевшим и по умершим от Covid во всём мире.
Бот здоровается с пользователем, пользователь вводит страну, если страна есть, то бот выдаёт статистику.
Также реализованы кнопки, нажав на каторые можно узнать статистику по конкретной стране.
Программа написана благодаря библиотекам: pyTelegramBotAPI-4.1.0, и COVID19Py, который установлены в PH при помощи pip.
Название бота и ключ, берётся у BotFather в телеграмм. Ключ вводится в  bot = telebot.TeleBot().
Свойства декораторов и функций которые использовались в программе можно почитать в документации на сайте https://pypi.org/.
Например location = covid19.getLatest() из библиотеки COVID19Py, определяет количество заблоевших и смертей по всему миру.
а location = covid19.getLocationByCountryCode(), определяет количество заболевших в определённой стране.
Таким образом реализован телеграмм бот.
